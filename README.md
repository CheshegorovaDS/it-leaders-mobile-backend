# Backend

Backend.

- Использовался фреймворк Spring boot
- Язык Kotlin
- В качестве Web Server’а выбрали Undertown
- База данных - Postgresql
- Для маппинга данных в БД использовали JPA

Использовали подход Controllers - Servers - Repositories.
